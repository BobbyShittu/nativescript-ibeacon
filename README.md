# NativeScript iBeacon
Let you range for beacons

### Install it
```
tns plugin add nativescript-ibeacon
```

### Use it
```typescript
import {NativescriptIbeacon} from 'nativescript-ibeacon';
let nativescriptIbeacon: NativescriptIbeacon = new NativescriptIbeacon();
```
